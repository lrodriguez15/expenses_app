# Personal Expenses App 

Expenses tracking app built with flutter 

- Flutter 3.3.10 
- Dart 2.18.6
- DevTools 2.15.0

Tested in Pixel 3a Android x64 Emulator. 

Styled for Android with Material and IOS with Cupertino.
